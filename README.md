# SMSBook

Easy social network like facebook without Internet

# Setup

## Use Npm
```Bash
$ npm install
```

## Use Yarn
```Bash
$ yarn install
```

# Run 

## React-Native

```bash
$ react-native run-android
```

## Npm

```bash
$ npm run android
```

## Yarn

```bash
$ yarn android
```
